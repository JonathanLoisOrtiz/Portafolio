# Jonathan J. Lois Ortiz - Interactive Portfolio

A modern one-page portfolio built with **HTML, CSS, and JavaScript**, based on the CV content.

## Features

- Interactive hero with animated typing effect
- Floating particle background and cursor glow effect
- Experience timeline with category filters
- Animated achievement counters
- 3D tilt cards for personal projects
- Dark/light theme toggle with saved preference
- Responsive layout for desktop and mobile

## Files

- `index.html` — semantic portfolio structure and CV content
- `styles.css` — visual design, responsive layout, and animations
- `script.js` — interactive behavior and UI effects
